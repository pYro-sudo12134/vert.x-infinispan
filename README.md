# File Storage Service

Сервис для менеджера томов с асинхронной обработкой файлов и возможностью миграции файлов.

| Технология            | Версия       | Назначение                                   |
|-----------------------|--------------|----------------------------------------------|
| Java                  | 17           | Язык программирования                        |
| Vert.x                | 4.5.13       | Реактивный фреймворк                         |
| Infinispan            | 14.0.29Final | Кластеризация и кэширование                  |
| JGroups               | -            | P2P общение в кластере                       |
| Apache Lucene         | 8.11.2       | Движок для поддержки индексаций в Infinispan |
| NFS                   | -            | Общее файловое хранилище                     |
| Docker/Docker Compose | -            | Контейнеризация                              |
